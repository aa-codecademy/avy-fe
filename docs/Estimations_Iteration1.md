# Avy Iteration 1 Estimations (Student Teams)

## Capacity Model

- Frontend only, mocked server/data, fixed design style
- No planned unit/integration/E2E testing in this iteration
- AI-assisted development is allowed
- 3 parallel teams, one team per epic, ~10 students each
- Student pace: 2h/day, working days only (5 days/week), 12 weeks
- Team capacity per epic: **60 team-days** (`10 x 2h x 5 x 12 / 20h`)
- Equivalent per epic: **12 weeks**

Note: Estimations include student-team delivery realities (collaboration, coordination, and manual verification effort).
